== Description ==
Firebase cloud messaging(FCM) for Topic.
Both Android and IOS devices are available
Post and Page Create and Update can trigger the Push Notification

**Plugin features:-** 

* Firebase cloud messaging(FCM) for Both Android and IOS
* Firebase cloud messaging(FCM) on post Create and Update
* Firebase cloud messaging(FCM) on page Create and Update


**How to use Plugin:-**

*Set up Firebase Project
* https://firebase.google.com/docs/cloud-messaging/concept-options
* After Install the plugin Save the server key to the plugin
